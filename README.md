# VWELDCORE Mobile Automation Bundle

A bundled Android + Termux automation infrastructure toolkit.

Includes multiple lightweight deployable infrastructure products for mobile automation workflows.

### 1. Termux Crypto Monitor
- Polygon RPC monitoring

- operational runtime infrastructure
